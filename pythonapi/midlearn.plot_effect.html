Plot MID component functions with plotnine.

Visualize the estimated main or interaction effect of a fitted MID model. This is a porting function for the R function midr::ggmid.mid().

Parameters:

  • term (str) ‚Äì The name of the component function (main effect or interaction term) to plot.

  • style ({'effect', 'data'}, default 'effect') ‚Äì The plotting style. ‚Äòeffect‚Äô plots the estimated component function as a line or a surface. ‚Äòdata‚Äô plots the specified data points (jittered for factor variables) with MID values represented by color.

  • theme (str or pt.color_theme or None, default None) ‚Äì The color theme to use for the plot.
